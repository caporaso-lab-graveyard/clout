Clout: Cloud-based Automated Testing

clout is a tool that facilitates the automated testing of software using Amazon's EC2 service as a testing platform. clout can work with any number of testing suites and programming languages.

clout is currently being used to test the development versions of the following bioinformatics projects on a nightly basis:

  • QIIME
  • PyCogent
  • biom-format
  • Primer Prospector
  • PyNAST

If you are using clout to test your software, we'd love to hear from you and add your project to the list!

For more information about clout, check out the GitHub project page.

Support

To get help with clout, report an issue, or request a feature, please create an issue on the clout GitHub issue tracker.
